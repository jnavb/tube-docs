# Introduction

Methods are the way we can call the functions of an object. To use a method type the function name preceded by the keyword `::`

# Example


<TubeCode>{`state\n::join with ', '\n`}</TubeCode>

<JSCode>{`const state = () => ([1, 2, 3])
`}</JSCode>

> "1, 2, 3"
